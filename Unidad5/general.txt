Data/
    Repositories/
    ListadoEntidad: IListadoEntidad(implementa)
    - List<clsEntidad>: listado con registros de tipo clsEntidad

    + List<clsEntidad> getListadoEntidad() - devuelve listado de misiones almacenado
    + clsEntidad getListadoEntidadById(int id) - devuelve registro de la lista en base al Id recibido (opcional)

(Proyecto MVC)
UI/
    Controllers/
        EntidadController : Controller - gestiona las peticiones del usuario y se comunica con el caso de uso para obtener los registros del listado
            - IUseCase _useCase; (inyectado)
            constructor EntidadController(IUseCase useCase) - asigna la instancia del caso de uso y la asigna al campo _useCase

            Index() - obtiene desde la llamada a getListadoEntidad() del caso de uso un dto listadoEntidadDTO con el listado de registros de tipo clsEntidad y lo pasará a la vista (puede llegarle una excepción si no se cumplen las condiciones establecidas en el caso de uso al hacer la llamada)

            [HttpPost]
            Index(int Id) - recibe el id por POST y obtiene desde la llamada al getListadoEntidadDTO(Id) del caso de uso un dto listadoEntidadDTO con el listado de registros de tipo clsEntidad y el registro seleccionado y lo pasará a la vista (puede llegarle una excepción si no se cumplen las condiciones establecidas en el caso de uso al hacer la llamada)
    Views/
        Index - mostrará lo que se pida en el ejercicio que deberá mostrar la vista
